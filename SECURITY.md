# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `zonetimeline-tui`, please report it responsibly. **Do not open a public issue.**

Instead, please use one of the following methods:

1. **GitHub Security Advisories** (preferred): Use the [Report a vulnerability](https://github.com/findyourexit/zonetimeline-tui/security/advisories/new) button on the Security tab of this repository.

2. **Email**: Contact the [maintainer](https://github.com/findyourexit) directly.

## What to Include

When reporting a vulnerability, please provide:

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any relevant configuration or environment details
- Suggested fix, if you have one

## Response

We will acknowledge receipt within 7 days and aim to provide a resolution or mitigation plan within 30 days. We will keep you informed of progress toward a fix and may ask for additional information.

## Supported Versions

Security fixes are applied to the latest release only. We recommend always running the most recent version.
